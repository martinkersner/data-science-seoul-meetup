Tournament

The tournament will be held in the following way:
  • All trained agents will compete with each other for several games during 1 minute.
  • Based on the win/loss ratio, the best 50 % of agents will be selected to the next round.
  • This process will be repeated until there is only one agent left.
  • Agents are allowed to learn during the competition.
  • If an agent takes more than 1 seconds to act, random move will be perfomed automatically.
